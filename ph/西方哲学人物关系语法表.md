# 西方哲学人物关系语法表

## 文档目的

本文件用于支持自动生成一个“西方哲学家关系网络”的网页应用。  
目标不是做百科全书，而是提供一份适合程序读取、适合可视化网络建模的工作底稿。

建议将本文件解析为两类数据：

1. `nodes`：人物节点
2. `edges`：人物之间或人物与传统之间的关系边

---

## 建议的数据模型

### 节点字段建议

- `id`：节点唯一标识
- `label`：显示名称
- `type`：`philosopher` / `tradition` / `context`
- `era`：时代分组
- `region`：主要活跃地区
- `course_role`：`core` / `secondary` / `bridge`
- `tags`：关键词数组

### 边字段建议

- `source`：起点节点 id
- `target`：终点节点 id
- `relation_type`：关系类型
- `relation_verb`：关系动词
- `statement`：完整关系句
- `weight`：建议可视化权重，1 到 5
- `period`：大致所属阶段
