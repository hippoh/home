# The Alethiometer Atlas

一个纯前端静态版《His Dark Materials》交互式人物关系网站，包含：

- 角色卡片浏览、搜索与筛选
- 顶部三册书模块勾选，联动角色、世界、关系和航路图
- Daemon、阵营、原生世界与经历过的世界
- 人物关系星图
- 多世界航路图
- 浅色蒸汽朋克、伦敦与牛津学院混合视觉风格

## 使用

最简单的方式是直接打开根目录下的 [hdm.html](/Users/hippoh/Desktop/code/homepage/dark/hdm.html)。

如果你想从终端里启动一个本地静态服务，也可以在项目目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://127.0.0.1:8000`。
